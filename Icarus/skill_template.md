
# Skill Markdown file template structure

## Folder Structure

- `skills/`         <- Skills Directory
  - `SKILL`         <- The skill folder. Replace "SKILL" with the actual skill name.
    - `SKILL.md`    <- MD file for LLMs. Contains what the skill does, parameters, etc.
    - `execute.py`  <- The actual execution code. The first version is built around python execution code but future iterations may either accept or be centered around Rust/C/C++
    - `config.json` <- JSON file for configs.
